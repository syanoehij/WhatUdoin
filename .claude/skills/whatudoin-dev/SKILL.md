# WhatUdoin 개발 오케스트레이터

메인 컨텍스트를 보호하기 위해 모든 분석·계획·지휘를 플래너 에이전트에게 위임한다.

## 실행

플래너 에이전트를 호출한다:

```
Agent(
  description: "WhatUdoin 개발 플래너",
  subagent_type: "planner",
  prompt: "사용자 요청: [사용자 요청 원문 그대로]"
)
```

플래너가 반환한 요약을 그대로 사용자에게 전달한다.
서버 재시작이 필요하다고 명시된 경우, 사용자에게 직접 요청한다.
